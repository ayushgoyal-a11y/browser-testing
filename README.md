# Steps to use debugger

1. Run `npm run start-tunnel` to start the tunnel
2. Copy the tunnel URL and paste it in the `src/debug.ts` file
3. Open a new terminal and Run `npm run debug` to start the local browser
4. Copy the socket URL from the terminal and use it for debugging session

Notes:

1. The socket URL will be different for each run
2. Close the tunnel when you are done debugging
